# Freelancehunt [Android]

Another one Freelancehunt.com client app.

```Kotlin```  ```Clean Architecture```
## Features
- **Feed**
  - List of user feed updates ```Cached```
- **Projects**
  - List of employers projects ```Cached```
  - Overview
    - About
    - Bids
    - Comments
  - Cached
- **Contests**
  - List of employers contests ```Cached```
  -  My contests ```for employers```
  - Overview
    - About
    - Comments
- **Freelancers**
  - All freelancers list
  - Overview
    - About
    - Reviews
- **Employers**
  - All employers list
  - Overview
    - About
    - Reviews
- **Messages**
  - All user threads list
  - Thread messages
  - Create thread
  - Send message
- **Bids**
  - My bids list ```for freelancers```
  - Add / Revoke / Reject / Choose winner bid
- **Notifications**
  - New feed events
  - New messages
